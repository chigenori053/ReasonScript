# Bottom Tool Window Contract

Status: Phase 3.5 DRAFT FOR ADOPTION

The Bottom Tool Window contains operational feedback that should not compete
with runtime reasoning inspection in the Right Inspector.

## Primary Tabs

- Problems: detailed diagnostics list and source navigation.
- Output: runtime print/output and runtime input/output operations.
- Logs: backend, API, workspace, artifact, or UI session logs when available.
- Tests: regression, baseline, and test results when available.

The Bottom Tool Window may be collapsible or resizable in a future shell, but
its primary tab set remains limited to Problems, Output, Logs, and Tests.
